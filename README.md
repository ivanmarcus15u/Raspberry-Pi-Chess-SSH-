# Raspberry-Pi-Chess-SSH-
Made this with ALOT of help with ChatGPT. First time publishing on GitHub.

Install these before running:
Python 3,
python-chess,
Stockfish engine

Features
Play full legal chess,
ASCII board display in terminal,
Choose AI strength (ELO 200–3000),
Undo moves,
Resign or quit anytime,
Engine plays realistic moves using Stockfish,
Automatic PGN generation for every game,
Works over SSH on low-power devices like Raspberry Pi Zero

At the end of every game:
Win/loss message (checkmate detected),
Full PGN printed in terminal,
PGN saved as a file for upload to chess sites like Lichess or Chess.com
